A CSS Sticky Footer

This is a page with a sticky footer consisting only of CSS and HTML.

It is glued to the bottom of the page, but stays below the content when the browser window is not high enough to show both the page and the footer.

See it at work by resizing the height of browser window.

The framework of the page consists of a 'page' div and a 'footer' div. Both divs have only one child div with class 'wrapper' with a top padding or top border of at least 1px to stop the top margins of nested elements from collapsing with the content and footer divs.

The page div is aqua colored, the footer div dark aqua. The nested wrapper div of the page div is blue, the nested wrapper div of the footer dark blue.

It is tested in all major browsers on Mac, PC and iOS. If it doesn't work in a certain browser on a certain system, it will be fixed.

See sticky-footer.css for the technical stuff.
